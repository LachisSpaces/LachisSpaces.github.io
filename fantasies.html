Lachis Fantasies
Glitch Garden
A 'tower defense' game inspired by the 'Plants vs. Zombies' game.
The images and sounds come from glitchthegame.com.
Space Defender
A shoot 'em up game inspired by the Space Invaders games.
It has over 100, increasinlgy difficult attack formations culminating in a bullet hell.
Protect the Core
This game was inspired by games like Pong and Breakout.
As far as I know, such a game has never been done before.
Space Ballet
This is a non-playable demo, showing some of the attack formations in my Space Defender game.
Enjoy how the invaders from space self destruct themselves in agony.
Number Guessing Game
This was the first game taught in the Unity developer course. Play against the computer, which tries to guess your number.
